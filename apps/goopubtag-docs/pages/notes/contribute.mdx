# Contribute

As can be seen in the list of [future improvements](/notes/future_improvements), there is plenty of opportunity to contribute to this project.

## How you can contribute

You can help out by:

- Raising issues when you encounter a bug
- Create PRs to fix raised issues

The library itself is a separate repository to the documentation. Improvements to the docs are also welcome!

- [goopubtag-docs](https://github.com/IainMcHugh/goopubtag-docs)
